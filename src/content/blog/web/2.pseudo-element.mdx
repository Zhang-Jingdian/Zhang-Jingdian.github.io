**Pseudo-elements** in CSS are used to style specific parts of an element's content. Unlike pseudo-classes, which target elements based on their state (like `:hover` or `:focus`), pseudo-elements allow you to apply styles to parts of an element, such as the first letter, first line, or to insert content before or after an element.

### Difference Between Pseudo-classes and Pseudo-elements
- **Pseudo-classes** (`:hover`, `:focus`, etc.): Target elements based on their state.
- **Pseudo-elements** (`::before`, `::after`, etc.): Target specific parts of an element's content.

### Common Pseudo-elements
Here are some frequently used pseudo-elements and their purposes:

1. **`::before` and `::after`**
   - Used to insert content **before** or **after** an element's actual content. This is commonly used for decorative elements or icons.
   ```css
   .button::before {
       content: '🌟';   /* Insert a star icon before the button */
       margin-right: 8px;
   }
   .button::after {
       content: '→';    /* Insert an arrow after the button */
       margin-left: 8px;
   }
   ```

2. **`::first-letter`**
   - Targets the **first letter** of an element. Useful for creating drop caps or highlighting the initial character.
   ```css
   p::first-letter {
       font-size: 2em;
       color: red;
       font-weight: bold;
   }
   ```

3. **`::first-line`**
   - Targets the **first line** of text in an element. Useful for emphasizing the first line in paragraphs.
   ```css
   p::first-line {
       font-weight: bold;
       color: blue;
   }
   ```

4. **`::selection`**
   - Styles the text **selected** by the user, such as changing the background color when highlighting text.
   ```css
   ::selection {
       background-color: yellow;
       color: black;
   }
   ```

### Example: Using Pseudo-elements
The following example demonstrates how to use `::before` and `::after` to insert content around a button, as well as `::selection` to change the appearance of selected text.

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Pseudo-element Example</title>
    <style>
        /* Basic button styling */
        .button {
            background-color: #007bff;
            color: white;
            padding: 10px 20px;
            border: none;
            border-radius: 5px;
            cursor: pointer;
            font-size: 16px;
            position: relative;
        }

        /* Insert an icon before the button text */
        .button::before {
            content: '🚀';   /* Insert a rocket icon */
            margin-right: 8px;
        }

        /* Insert an arrow after the button text */
        .button::after {
            content: ' ➜';
            margin-left: 8px;
        }

        /* Style for text selection */
        ::selection {
            background-color: yellow; /* Highlighted text background */
            color: black;             /* Highlighted text color */
        }
    </style>
</head>
<body>
    <button class="button">Click Me</button>
    <p>Try selecting this text to see the custom selection style.</p>
</body>
</html>
```

### Explanation
1. **`::before` and `::after`**:
   - Inserts a rocket icon before the button's text and an arrow after it using the `content` property.
   - Adjusts spacing using `margin-right` and `margin-left`.

2. **`::selection`**:
   - Changes the background color and text color when users select text on the page.

### What You’ll See
- The button will have a rocket icon before its text and an arrow after it.
- When you select any text on the page, the highlighted background changes to yellow with black text.

### Conclusion
Pseudo-elements are incredibly useful in web design, allowing you to enhance the appearance of elements without modifying the HTML structure. They are especially helpful for adding decorative content, customizing text, or improving user experience through interactive styles.
