a gradient is a smooth transition between two or more colors, commonly used as backgrounds or effects. 

CSS gradients are created using the background-image property and can be linear or radial.

### 1. **Linear Gradient**

A **linear gradient** creates a transition between colors along a straight line. You can control the direction or angle of the line.

#### Syntax:
```css
background-image: linear-gradient(direction, color1, color2, ...);
```

- `direction`: The direction the gradient follows. It can be an angle (e.g., `45deg`) or a keyword (e.g., `to right`, `to top`).
- `color1`, `color2`, etc.: The colors to be used in the gradient.

<Collapse title=" Examples">

1. **Gradient from left to right:**
   ```css
   background-image: linear-gradient(to right, red, yellow);
   ```
   This creates a gradient that goes from **red** on the left to **yellow** on the right.

2. **Gradient at an angle:**
   ```css
   background-image: linear-gradient(45deg, red, yellow);
   ```
   This creates a gradient at a **45-degree angle** starting with **red** and ending with **yellow**.

3. **Multiple color stops:**
   ```css
   background-image: linear-gradient(to right, red, orange, yellow);
   ```
   This creates a gradient that transitions from **red** to **orange** to **yellow** as it moves from left to right.
</Collapse>
### 2. **Radial Gradient**

A **radial gradient** creates a circular or elliptical gradient that radiates outward from a central point.

#### Syntax:
```css
background-image: radial-gradient(shape size at position, color1, color2, ...);
```

- `shape`: The shape of the gradient (`circle` or `ellipse`).
- `size`: The size of the gradient (e.g., `closest-side`, `farthest-corner`).
- `position`: The starting point (default is `center`).

<Collapse title=" Examples">

1. **Circular gradient (default):**
   ```css
   background-image: radial-gradient(red, yellow);
   ```
   This creates a **circular** gradient with **red** in the center and **yellow** at the edges.

2. **Elliptical gradient:**
   ```css
   background-image: radial-gradient(ellipse, red, yellow);
   ```
   This creates an **elliptical** gradient starting with **red** at the center and transitioning to **yellow**.

3. **Gradient with custom position:**
   ```css
   background-image: radial-gradient(circle at top left, red, yellow);
   ```
   This creates a **circular gradient** that starts at the **top-left** of the element.
</Collapse>
### 3. **Repeating Gradients**

A **repeating gradient** will repeat the gradient pattern indefinitely. You can use either linear or radial gradients for this.

#### Syntax for linear:
```css
background-image: repeating-linear-gradient(direction, color1, color2, ...);
```

#### Example:
```css
background-image: repeating-linear-gradient(45deg, red, yellow 20%, green 40%);
```
This creates a **45-degree gradient** that repeats, transitioning from **red** to **yellow** (20% of the width) and then to **green** (40%).

### 4. **Multiple Color Stops**

You can define multiple color stops at specific points to create complex gradient effects.

#### Example:
```css
background-image: linear-gradient(to right, red 0%, yellow 50%, green 100%);
```
- **Red** starts at 0%, transitions to **yellow** at 50%, and finally reaches **green** at 100%.

## Example:

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <style>
    .gradient-box {
      width: 300px;
      height: 300px;
      /* Linear gradient from top left to bottom right */
      background-image: linear-gradient(45deg, red, yellow);
    }
    .radial-box {
      width: 300px;
      height: 300px;
      /* Radial gradient */
      background-image: radial-gradient(circle, red, yellow);
    }
  </style>
</head>
<body>
  <div class="gradient-box"></div>
  <div class="radial-box"></div>
</body>
</html>
```

### Summary of Key Points:

- **Linear gradient**: Gradients transition in a straight line (with customizable directions or angles).
- **Radial gradient**: Gradients radiate outward from a center point (either circular or elliptical).
- **Repeating gradient**: A gradient pattern that repeats over the entire element.
- **Multiple color stops**: You can specify several colors at defined points in the gradient.
