In CSS, the `@` symbol is used to denote **at-rules**, which are special instructions that provide additional functionality or behavior for your stylesheets. Some common at-rules include:

1. **@import**: Used to import external CSS files into the current stylesheet.
   ```css
   @import url("styles.css");
   ```

2. **@media**: Defines different styles for different devices or screen sizes (responsive design).
   ```css
   @media screen and (max-width: 600px) {
     body {
       background-color: lightblue;
     }
   }
   ```

3. **@font-face**: Allows you to define custom fonts to use in your website.
   ```css
   @font-face {
     font-family: "MyCustomFont";
     src: url("fonts/mycustomfont.woff2") format("woff2");
   }
   ```

4. **@keyframes**: Defines animations by specifying a sequence of keyframes.
   ```css
   @keyframes example {
     from {
       background-color: red;
     }
     to {
       background-color: blue;
     }
   }
   ```

5. **@supports**: Used to apply styles only if a certain feature or property is supported by the browser.
   ```css
   @supports (display: grid) {
     .container {
       display: grid;
     }
   }
   ```

These at-rules enable more advanced and flexible styling in CSS.
