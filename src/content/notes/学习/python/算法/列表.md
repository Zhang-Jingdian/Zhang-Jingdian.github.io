#算法 
# 1. int 转 list

```python
num = 123
nums = list(map(int, str(num)))
```
# 2. list(int) 转 int

```python
nums = [1, 2, 3]
num = int(''.join(map(str, nums)))

def lst_int(nums):
    return int(''.join(map(str, nums)))
```

# 3. 列表特性

比较大小的时候，不管长度如何，依次比较到第一个元素不相等的位置

比如 [1, 2, 3] < [2, 3] 因为在比较 1 < 2 的时候就终止。

嵌套列表推导：展平二维数组

```python
nums = [e for row in matrix for e in row]
```
