命令行前面有 `(.venv)`，这是因为你**激活了 Python 的虚拟环境（virtualenv）**。

---

## 解释

- `(.venv)` 是 Python 虚拟环境的默认提示符，表示你当前在一个隔离的 Python 环境里。
- 这样做的好处是：**不会污染全局 Python 包，依赖管理更安全**。
- 你在终端输入
- `source .venv/bin/activate` 或 `python -m venv .venv` 后，
- 命令行前面就会出现 `(.venv)`。

---

## 退出虚拟环境

如果你想回到“普通”shell，只需输入：

```bash
deactivate
```
